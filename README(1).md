# Fashion-MNIST-KNN-Classifier
A KNN-based machine learning project that classifies fashion images using the Fashion-MNIST dataset, with support for user input image prediction.

# 👕 Fashion Image Classifier using KNN

## 📌 Description
This project is a Fashion Image Classifier built using the K-Nearest Neighbors (KNN) algorithm. It uses the Fashion-MNIST dataset to classify images into categories like shirts, shoes, and bags. The model is trained on pixel features and supports user input to predict custom images.

---

## 🚀 Features
- Image classification using KNN algorithm  
- Uses Fashion-MNIST dataset  
- Data preprocessing (normalization)  
- PCA for dimensionality reduction  
- Accuracy evaluation and confusion matrix  
- User input image prediction feature  

---

## 🧠 Technologies Used
- Python  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 📂 Dataset
- Dataset: Fashion-MNIST  
- Images: 28×28 grayscale images  
- Classes: 10 categories (T-shirt, Trouser, Sneaker, Bag, etc.)

---

## ⚙️ How It Works
1. Load dataset  
2. Preprocess images (flatten + normalize)  
3. Train KNN model  
4. Select best K value  
5. Evaluate model performance  
6. Predict new images (user input)

---

## ▶️ How to Run
1. Clone the repository  
2. Place dataset files in the project folder  
3. Run the Python file or Jupyter Notebook  
4. Upload or provide an image  
5. View prediction result  

---

## 📊 Output
- Accuracy results  
- Confusion matrix  
- Predicted image category  

---

## 🎯 Conclusion
This project demonstrates how the KNN algorithm can be used for image classification. It provides a simple and effective approach with support for real-time user input prediction.
